## About The Project

This project is built on Angular.js and Typescript as frontend and Express,Node.js as backend. The project has features such as Authentication and Authorization, Creating,Editing,deleting and viewing the posts based on the authorization and uploading images from the frontend. The project has database connected to the MongoDB 
and is yet to be deployed on AWS using ElasticBeanStalk and S3.
